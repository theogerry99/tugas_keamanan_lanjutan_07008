# 🔐 Sistem Login & Registrasi Sederhana dengan PHP

Sistem ini dibuat menggunakan PHP dan MySQL untuk fitur login dan registrasi pengguna. Cocok digunakan sebagai latihan dasar pemrograman web dan keamanan.

## ✨ Fitur

- Registrasi pengguna dengan penyimpanan password menggunakan `password_hash()`
- Login dengan validasi aman menggunakan `password_verify()`
- Logout untuk mengakhiri sesi pengguna
- Dashboard sebagai halaman setelah login
- Koneksi database menggunakan PDO untuk menghindari SQL injection

## 🛠 Teknologi

- PHP 7.4+
- MySQL
- HTML + CSS dasar
- PDO untuk koneksi database
